# Security Policy

This project documents controlled academic network-security lab work. It is not intended to be run against third-party networks or production systems.

## Scope

- Use only in isolated lab environments where you have authorization.
- Do not publish credentials, raw packet captures, private course artefacts, or licensed network images.
- Treat all included configurations and commands as educational examples.

## Reporting issues

If you notice leaked secrets, copyrighted artefacts that should not be included, or unsafe publication content, open an issue or contact the repository owner privately.
